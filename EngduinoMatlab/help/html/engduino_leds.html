Engduino LEDs

Engduino LEDs

setLed Turn on/off the small green LED on Engduino
setLeds set individual colours of all the 16 LEDs on Engduino
setLedsAll set one colour for all the 16 LEDs on Engduino
setLedsExact set the colour for one of the 16 LEDs on Engduino, other LEDs will be turned off
setLedsOne set the colour for one of the 16 LEDs on Engduino, other LEDs will remain in previous state
setLedsB set individual colours and brightness for all 16 LEDs on Engduino
setLedsAllB set the colour and brightness of all 16 LEDs on Engduino

LED Colour Properties

Colours are defined in Engduino object e

e.COLOR_RED Set Engduino LED colour as RED, value=0
e.COLOR_GREEN Set Engduino LED colour as GREEN, value=1
e.COLOR_BLUE Set Engduino LED colour as BLUE, value=2
e.COLOR_YELLOW Set Engduino LED colour as YELLOW, value=3
e.COLOR_MAGENTA Set Engduino LED colour as MAGENTA, value=4
e.CYAN Set Engduino LED colour as CYAN, value=5
e.WHITE Set Engduino LED colour as WHITE, value=6
e.OFF Turn off Engduino LED, value=7
