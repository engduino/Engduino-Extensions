Functions Reference

Setup and Configuration

engduino Connect to Engduino hardware

Engduino LEDs

setLed Turn on/off the small green LED on Engduino
setLeds set individual colours of all the 16 LEDs on Engduino
setLedsAll set one colour for all the 16 LEDs on Engduino
setLedsExact set the colour for one of the 16 LEDs on Engduino, other LEDs will be turned off
setLedsOne set the colour for one of the 16 LEDs on Engduino, other LEDs will remain in previous state
setLedsB set individual colours and brightness for all 16 LEDs on Engduino
setLedsAllB set the colour and brightness of all 16 LEDs on Engduino

Engduino Button

getButton Get the status of the button on Engduino whether it is pressed or released

Engduino Sensors

getAccelerometer Get the xyz value of accelerometer on Engduino
getTemperature Get the temperature value in celcius from Engduino
getLight Measure light intensity from light sensor on Engduino
getMagnetometer Get the xyz value of magnetometer on Engduino

Read and Write Data

analogRead Reads the value from the specified analog pin
analogWrite Writes an analog value (PWM wave) to a pin
digitalRead Reads the value from the specified digital pin
digitalWrite Write a HIGH or a LOW value to a digital pin
pinMode Configures the specified pin to behave either as an input or an output
pinModeA Configures the specified pin to behave either as an input or an output
